## 📌 Project Overview

<div align="center">

🚦 This project classifies **traffic signs from images** using a
**Convolutional Neural Network (CNN)**.

📷 Upload an image → Model predicts the traffic sign instantly.

💡 Useful for **self-driving cars, driver assistance systems, and road safety applications**.

</div>

---

## 🎯 Objective

* Automatically recognize traffic signs
* Improve road safety using AI
* Demonstrate CNN-based image classification
* Deploy model as an interactive web app

## 🚀 How It Works

1. Upload a traffic sign image
2. Image is preprocessed (resized, normalized)
3. CNN model analyzes features
4. Predicted traffic sign is displayed

---

## 📊 Model Insights

* Learns visual patterns from traffic sign images
* Uses convolution layers to extract features
* Provides fast and accurate predictions
* Suitable for real-time applications
